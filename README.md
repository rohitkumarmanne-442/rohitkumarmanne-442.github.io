# Rohit Kumar Manne — Portfolio

A unique, single-page portfolio for **Rohit Kumar Manne**, Agentic AI Engineer.

🔗 **Live:** https://rohitkumarmanne-442.github.io

## Features
- **Glassmorphism** UI with an animated aurora background
- Rich **scroll effects** — hero parallax, a horizontal pinned-scroll section, an infinite velocity-reactive marquee, and word-by-word title reveals
- **Live GitHub** repositories pulled from the GitHub API
- A built-in **AI chat agent** (bottom-right) trained on the résumé — answers questions about experience, skills, and projects with zero backend

## Stack
Pure **HTML + CSS + vanilla JavaScript** — no build step, no dependencies. Just open `index.html`.

## Structure
| File | Purpose |
|------|---------|
| `index.html` | Markup & sections |
| `styles.css` | Glassmorphism + animations |
| `script.js` | Data, scroll effects, GitHub fetch, AI agent |
